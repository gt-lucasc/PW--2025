O que é um banco de dados? 

- Qualquer tipo de dados armazenados e organizados para futura consulta;

LGPD - Usuários deverá ter acesso aos seus dados para fazer o que quiser;

Tipos de banco de dados 

-Relacionais - índice, chaves números e colunas, mais organizados e lentos.

-Não relacionais - 
São desorganizados e mais rápidos.

Comandos básicos de SQL

CREATE TABLE;
cria uma tabela.

ALTER TABLER;
altera a estrutura e modifica o banco de dados.

DROP TABLE;
remove componentes de uma tabela do banco de dados.

CONSTRANTS;
estabelece regras para os dados

SELECT;
consultar dados.

INSERT;
algum dado.

UPDATE;
para modificar dados.

FROM;
usado para especificar a tabela.

WHERE;
usado para adicionar condições às consultas.


DELETE;
remove registros do banco de dados.
